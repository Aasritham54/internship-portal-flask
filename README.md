# Internship Portal – Flask Web Application

This is a full-stack internship portal built using Flask and SQLite.

## Features
- User registration and login
- Password hashing
- Role-based dashboards
- Internship listings
- Clean UI using Flowbite
- SQLite database

## Tech Stack
- Python (Flask)
- HTML, Tailwind CSS, Flowbite
- SQLite
- SQLAlchemy
- Git & GitHub

## How to Run

git clone https://github.com/Aasritham54/internship-portal-flask.git
cd internship-portal-flask
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py

Open:
http://127.0.0.1:5000
